2. 핵심 개념 정리

사인 (Sine)

단위 원 위의 점 P(x, y)에 대해, 동경 OP가 나타내는 각을 θ라 할 때, sin(θ) = y 입니다. (높이)

주기: 2π, 최댓값: 1, 최솟값: -1

코사인 (Cosine)

단위 원 위의 점 P(x, y)에 대해, 동경 OP가 나타내는 각을 θ라 할 때, cos(θ) = x 입니다. (밑변)

주기: 2π, 최댓값: 1, 최솟값: -1

탄젠트 (Tangent)

tan(θ) = y/x 입니다. (기울기)

주기: π, 점근선: x = nπ + π/2
